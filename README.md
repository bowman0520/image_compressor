# 本地图片压缩工具

一个简单实用的本地图片压缩程序，支持 JPG、JPEG、PNG、WEBP 格式。通过按比例缩小分辨率或固定宽高 + 质量压缩来减小图片体积。

## 功能特性

- 支持 JPG / JPEG / PNG / WEBP 图片格式
- 三种压缩方式：按比例缩放、固定宽度、固定高度
- 按比例缩放：自定义缩放比例（0.1-1.0，默认 50%）
- 固定宽度：指定目标宽度，高度按原图比例自动计算
- 固定高度：指定目标高度，宽度按原图比例自动计算
- JPG 和 WEBP 可自定义压缩质量（1-100，默认 100 即最高质量）
- PNG 自动转为 palette 模式或使用最高压缩级别，尽量压缩体积
- 如果压缩后体积反而更大，自动保留原图，确保不会越压越大
- 压缩图片输出到原文件夹下的 `compressed` 子目录，不覆盖原图
- 压缩后文件名前缀加 `compressed_`
- 支持 Windows 中文路径
- 终端显示每张图片压缩前后大小及压缩百分比
- 最后输出汇总统计：压缩数量、节省空间

## 安装

```bash
# 建议使用虚拟环境
python -m venv venv

# Windows 激活虚拟环境
venv\Scripts\activate

# Linux / macOS 激活虚拟环境
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

## 使用方法

```bash
python compressor.py
```

运行后按提示操作：

1. 输入图片所在的文件夹路径
2. 选择压缩方式：
   - 1 = 按比例缩放（默认）
   - 2 = 固定宽度（输入目标宽度 px，高度按比例自动计算）
   - 3 = 固定高度（输入目标高度 px，宽度按比例自动计算）
3. 根据选择的压缩方式输入缩放比例或目标宽高
4. 输入 JPG/WEBP 的压缩质量（1-100，直接回车使用默认值 100）
5. 程序自动扫描、压缩并输出结果

## 参数说明

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 压缩方式 | 1/2/3 | 1 | 1=按比例缩放，2=固定宽度，3=固定高度 |
| 缩放比例 | 0.1-1.0 | 0.5 (50%) | 方式1时使用，按比例缩小分辨率，1.0 为不缩放 |
| 固定宽度 | >0 (px) | - | 方式2时使用，指定目标宽度，高度按比例自动计算 |
| 固定高度 | >0 (px) | - | 方式3时使用，指定目标高度，宽度按比例自动计算 |
| 压缩质量 | 1-100 | 100 | JPG/WEBP 质量参数，100 为最高质量 |

## 输出说明

- 压缩后的图片保存在原文件夹下的 `compressed` 目录中
- 文件名格式：`compressed_原文件名.扩展名`
- 原始图片不会被修改或删除

## 压缩策略

| 格式 | 策略 |
|------|------|
| JPG/JPEG | 缩放 + 指定 quality + optimize |
| WEBP | 缩放 + 指定 quality + method=6 (最高压缩) |
| PNG (无透明) | 缩放 + 转 palette 模式 + compress_level=9 |
| PNG (有透明) | 缩放 + 保留 RGBA + optimize + compress_level=9 |
